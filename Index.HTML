About Space Invaders

Made by BGU - SISE students Yuval and Idan

This classic arcade-style game challenges players to defend Earth from alien invaders.

Game features include different enemy types, power-ups, and increasing difficulty levels.

This version was created as an educational project.

Game controls:

  • Left/Right Arrow Keys: Move spaceship
  • Space Bar: Shoot
